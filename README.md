# Proyecto Web Social

## Descripción
**Proyecto Web Social** es una plataforma web que permite a los usuarios interactuar, compartir contenido y conectarse con otras personas. Está diseñado con enfoque en la experiencia de usuario, seguridad y escalabilidad.

Este proyecto es parte de los trabajos académicos de desarrollo web y tiene como objetivo aprender y aplicar tecnologías modernas en un entorno realista.

---

## Funcionalidades

- Registro y autenticación de usuarios
- Creación y gestión de perfiles
- Publicaciones, comentarios

---

## Tecnologías utilizadas

- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend:** PHP
- **Base de datos:** MySQL
- **Control de versiones:** Git y GitHub
